# Face Detection and Recognition of Characters from the Scooby-Doo Animated Series

This project implements a face detection and recognition system for characters from the *Scooby-Doo* animated series using classical computer vision techniques. The detection module is based on Histogram of Oriented Gradients (HOG) descriptors, a multi-scale Sliding Window approach, Non-Maximum Suppression (NMS), and Intersection over Union (IoU) for evaluation.


## Features

- Multi-scale face detection using HOG descriptors
- Sliding Window search strategy
- Non-Maximum Suppression (NMS)
- IoU-based evaluation
- Face recognition using HOG-based feature extraction
- Detector optimization through classifier specialization and score calibration

## Technologies

- Python
- OpenCV
- scikit-learn
- scikit-image
- NumPy
